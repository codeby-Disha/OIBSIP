# Python Programming Internship Projects

## Internship Overview

This repository contains the projects I completed during my **Python Programming Internship at OASIS INFOBYTE**. The internship focused on strengthening my core Python concepts through hands-on, practical projects. Over the course of this internship, I worked on **five Python-based projects**, each designed to improve my understanding of programming logic, problem-solving, and real-world application development.

The projects included:

* Voice Assistant Using Python
* BMI Calculator
* Random Password Generator
* Weather Application
* Browser-Based Chat Application

Each project is kept simple yet functional, focusing on clarity, core concepts, and clean implementation. Detailed explanations are provided below.


## 1. Voice Assistant Using Python

The Voice Assistant project is an interactive Python application that responds to voice commands. It uses speech recognition to understand user input and text-to-speech to respond verbally. The assistant can perform tasks such as telling the current date and time and searching queries on the web.

**Key Concepts Learned:**

* Speech recognition and text-to-speech conversion
* Working with Python libraries like `SpeechRecognition`, `pyttsx3`, `datetime`, and `webbrowser`
* Handling user commands and conditional logic
* Building interactive and automation-based applications

---

## 2. BMI Calculator

The BMI Calculator is a simple Python program that calculates Body Mass Index based on user-provided height and weight. The program categorizes the BMI result and includes proper error handling to manage invalid inputs.

**Key Concepts Learned:**

* Taking and validating user input
* Applying mathematical formulas in Python
* Using exception handling such as `ValueError` and `ZeroDivisionError`
* Writing clean and user-friendly console programs

---

## 3. Random Password Generator

This project generates secure random passwords based on user-defined preferences. The user can choose the password length and the type of characters to include, such as letters, digits, symbols, or a combination of all.

**Key Concepts Learned:**

* Using Python libraries like `random` and `string`
* Implementing user-choice-based logic
* Generating random data securely
* Understanding basic security concepts related to passwords

---

## 4. Weather Application

The Weather Application fetches real-time weather data using external APIs. API URLs and sensitive data are stored securely in a `.env` file, which is excluded from version control using `.gitignore`. The project was developed and executed inside a virtual environment.

**Key Concepts Learned:**

* Working with APIs using the `requests` library
* Environment variable management using `python-dotenv`
* Securing sensitive data with `.env` and `.gitignore`
* Using virtual environments in VS Code
* Reading and processing API responses

---

## 5. Browser-Based Chat Application

This is a client-server-based chat application developed using Python socket programming. It consists of two files: `server.py` and `client.py`, enabling real-time communication between the client and server.

**Key Concepts Learned:**

* Networking fundamentals and socket programming
* Client-server architecture
* Real-time message exchange
* Managing multiple Python files in a single project
* Writing technical documentation using README files
